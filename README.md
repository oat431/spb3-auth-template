# Spring Boot 3 Authentication Template for Monolith Applications
This template is a monolith application that use role based access control for authentication and authorization. 

### Features that gonna be implemented
* [ ] Sign up (email, username and password)
* [ ] Sign in (access token and refresh token)
* [ ] Sign out (revoke refresh token)
* [ ] Email verification
* [ ] Password reset
* [ ] Password change
* [ ] User management (admin only)
